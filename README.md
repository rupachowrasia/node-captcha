# NodeJS Captcha


## Installation
It requires [Node.js](https://nodejs.org/) v8+ to run.
Install the dependencies and devDependencies and start the server.
```sh
$ git clone https://github.com/rupachowrasia/node-captcha.git
$ cd node-captcha
$ npm install
$ npm start
```
